🎨 HTML e CSS

O que é HTML?

HTML significa HyperText Markup Language (Linguagem de Marcação de Hipertexto). Foi criado por Tim Berners-Lee em 1991, quando ele trabalhava no CERN (laboratório de física na Suíça). O HTML não é uma linguagem de programação — é uma linguagem de marcação, usada para estruturar o conteúdo das páginas web.

Hoje, o HTML é mantido pela W3C (World Wide Web Consortium) e pelo WHATWG. A versão atual é o HTML5, lançado em 2014, que trouxe suporte a vídeo, áudio, canvas e muito mais.

O que é CSS?

CSS significa Cascading Style Sheets (Folhas de Estilo em Cascata). Foi criado por Håkon Wium Lie em 1994 e se tornou recomendação do W3C em 1996. O CSS define como os elementos HTML devem ser exibidos visualmente: cores, fontes, tamanhos, posicionamento, etc.

A versão atual é o CSS3, que trouxe animações, gradientes, flexbox, grid e muito mais.

A diferença entre HTML e CSS

Característica HTML CSS
O que faz? Estrutura o conteúdo Estiliza o conteúdo
Exemplo Cria parágrafos, títulos, listas Define cores, fontes, tamanhos
Extensão do arquivo .html .css
Analogia O esqueleto de uma casa A pintura e decoração da casa

Exemplo de HTML

<!DOCTYPE html>
<html>
<head>
  <title>Minha Página</title>
</head>
<body>
  <h1>Olá, Mundo!</h1>
  <p>Esta é a minha primeira página web.</p>
  <ul>
    <li>Item 1</li>
    <li>Item 2</li>
  </ul>
</body>
</html>

Exemplo de CSS

/* Estilizando elementos */
body {
  background-color: #f4f4f4;
  font-family: Arial, sans-serif;
}

h1 {
  color: #e94560;
  font-size: 32px;
}

p {
  line-height: 1.6;
  color: #333;
}

Principais tags HTML

  • <h1> a <h6> — Títulos e subtítulos
  • <p> — Parágrafo de texto
  • <a> — Link (âncora)
  • <img> — Imagem
  • <ul> / <ol> / <li> — Listas
  • <table> — Tabela
  • <form> — Formulário
  • <div> — Divisão/container
  • <header>, <nav>, <main>, <footer> — Estrutura semântica (HTML5)
