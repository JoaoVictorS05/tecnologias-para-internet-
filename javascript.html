🌐 JavaScript

Como surgiu o JavaScript?

O JavaScript foi criado por Brendan Eich em apenas 10 dias em maio de 1995, enquanto trabalhava na empresa Netscape Communications. O objetivo era criar uma linguagem de script simples para tornar as páginas web interativas.

Inicialmente foi chamado de Mocha, depois LiveScript, e finalmente renomeado para JavaScript como estratégia de marketing, para aproveitar a popularidade do Java na época. Apesar do nome, JavaScript e Java são linguagens completamente diferentes!

Em 1997, o JavaScript foi padronizado pela ECMA International como ECMAScript. Hoje, é a única linguagem de programação que roda nativamente em todos os navegadores web.

Características do JavaScript

  • Linguagem interpretada, executada diretamente no navegador
  • Orientada a eventos (cliques, digitações, etc.)
  • Tipagem dinâmica (não precisa declarar o tipo das variáveis)
  • Pode ser usada no front-end (navegador) e no back-end (Node.js)
  • Linguagem mais popular do mundo segundo o Stack Overflow
  • Suporte a programação funcional e orientada a objetos

Exemplo de código JavaScript

// Olá Mundo em JavaScript
console.log("Olá, Mundo!");

// Variáveis
let nome = "Carlos";
let idade = 22;
alert("Bem-vindo, " + nome + "!");

// Função
function somar(a, b) {
  return a + b;
}
console.log(somar(5, 3)); // Resultado: 8

Onde JavaScript é utilizado?

  • Front-end Web: animações, formulários, menus interativos
  • Back-end: servidores com Node.js
  • Aplicativos Mobile: React Native
  • Jogos Web: usando Canvas e WebGL
  • Extensões de Navegador: Chrome, Firefox
  • Aplicações Desktop: Electron (VSCode é feito em JS!)
