Curiosidades sobre os Logos

  • O logo do Python mostra duas cobras entrelaçadas, uma azul e uma amarela.
  • O logo do JavaScript é simplesmente as letras "JS" em um fundo amarelo, criado pela comunidade.
  • O logo do Java é uma xícara de café fumegante, referência ao café de Java (Indonésia).
  • O logo do HTML5 é um escudo laranja com o número 5, representando força e modernidade.
  • O logo do CSS3 é um escudo azul, complementar ao HTML5.
  • O logo do Swift da Apple tem a forma de um pássaro (swift = andorinha em inglês).
