⚙️ Linguagem C e C++

Como surgiu a linguagem C?

A linguagem C foi criada por Dennis Ritchie entre 1969 e 1973, nos Bell Laboratories (laboratórios da AT&T) nos Estados Unidos. Ela foi desenvolvida para reescrever o sistema operacional Unix, que antes era escrito em Assembly.

O nome "C" vem do fato de que ela foi derivada de uma linguagem anterior chamada B, que por sua vez veio do BCPL. C trouxe o equilíbrio perfeito entre eficiência (próxima do hardware) e legibilidade (fácil para humanos).

Em 1978, Dennis Ritchie e Brian Kernighan publicaram o livro "The C Programming Language", que se tornou a referência mundial para aprender C.

Como surgiu o C++?

O C++ foi criado por Bjarne Stroustrup em 1983, também nos Bell Labs. Ele queria adicionar recursos de Orientação a Objetos ao C. O nome "C++" é uma piada de programador: em C, o operador "++" incrementa um valor, então "C++" significa "C melhorado".

Características do C

  • Linguagem de baixo nível (próxima ao hardware)
  • Muito rápida e eficiente
  • Controle manual da memória (ponteiros)
  • Base para quase todas as linguagens modernas
  • Usada onde velocidade é crítica

Características do C++

  • Tudo que C tem, mais Orientação a Objetos
  • Templates e programação genérica
  • Standard Template Library (STL)
  • Muito usada em jogos, sistemas embarcados e softwares de alto desempenho

Exemplo de código em C

#include <stdio.h>

int main() {
  printf("Olá, Mundo!\n");

  int i;
  for (i = 1; i <= 5; i++) {
    printf("Número: %d\n", i);
  }
  return 0;
}

Por que aprender C?

Mesmo sendo uma linguagem antiga, C ainda é muito importante. Aprender C ajuda a entender como os computadores funcionam por dentro: memória, ponteiros, registradores. Quem entende C aprende qualquer outra linguagem com muito mais facilidade!
