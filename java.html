☕ Java

Como surgiu o Java?

Java foi criado por James Gosling e sua equipe na empresa Sun Microsystems no início dos anos 1990. O projeto começou em 1991 com o codinome "Oak" (carvalho, em inglês), mas foi renomeado para Java em 1995 — nome inspirado no café da ilha de Java, na Indonésia.

O lema do Java é "Write Once, Run Anywhere" (Escreva uma vez, execute em qualquer lugar). Isso significa que um programa Java pode rodar em qualquer sistema operacional — Windows, Mac, Linux — sem precisar ser reescrito.

Isso é possível graças à JVM (Java Virtual Machine), uma máquina virtual que interpreta o código Java compilado (chamado de bytecode).

Características do Java

  • Fortemente orientado a objetos
  • Multiplataforma graças à JVM
  • Tipagem estática (é necessário declarar o tipo das variáveis)
  • Gerenciamento automático de memória (Garbage Collector)
  • Muito seguro e robusto
  • Amplamente usado em grandes empresas e sistemas bancários

Exemplo de código Java

// Olá Mundo em Java
public class OlaMundo {
  public static void main(String[] args) {
    System.out.println("Olá, Mundo!");

    // Variáveis
    String nome = "Maria";
    int idade = 25;
    System.out.println("Nome: " + nome + ", Idade: " + idade);
  }
}

Onde Java é utilizado?

  • Sistemas Bancários: muito presente em bancos e financeiras
  • Android: por muitos anos foi a principal linguagem para apps Android
  • Sistemas Corporativos: ERPs, CRMs, sistemas de grande escala
  • Servidores Web: com frameworks como Spring e Jakarta EE
  • Big Data: ferramentas como Hadoop são escritas em Java
  • Games: o famoso Minecraft foi criado em Java!

Java vs JavaScript

Característica Java JavaScript
Criado em 1995 (Sun Microsystems) 1995 (Netscape)
Tipagem Estática (forte) Dinâmica (fraca)
Onde roda JVM (qualquer sistema) Navegador / Node.js
Uso principal Sistemas corporativos, Android Web, front-end, back-end
Relação Nenhuma! Apenas o nome é parecido.
